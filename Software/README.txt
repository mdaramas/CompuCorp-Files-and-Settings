This folder includes the software that is used for Drupal 7 and CiviCRM 4.7.

drupal-7.54.tar.gz file must be extracted  and then copy (drupal-7.54 folder) its contents to /usr/share/nginx/html/drupal folder.
civicrm-4.7.16-drupal.tar.gz file must be extracted and then copied (civicrm folder) under the /usr/share/nginx/html/drupal/sites/all/modules folder.
certbot-auto file (used for the LetsEncrypt certificates) can be copied under /home/ec2-user/Source folder.
composer file (which is used to install Drush 8) can be copied under the /home/mdaramas user.
