This folder contains Nginx conf file along with the DHparam certificate file and the conf file for the php-fpm module and php.ini file.

nginx.conf file can be copied over the existing /etc/nginx/nginx.conf file.
www.conf file can be copied under /etc/php-fpm.d folder.
dhparams.pem can be copied under /etc/nginx/dhparam-key folder.
php.ini file can be copied over the existing /etc/php.ini.
